## 🌟 Overview

This project is a high-performance, modern portfolio website designed to present a curated collection of my best work. It is built with a focus on speed, accessibility, and a clean developer experience.

### 🛠 Tech Stack

- **Frontend:** [Astro](https://astro.build/) (Static Site Generator)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Backend:** [FastAPI](https://fastapi.tiangolo.com/) (Python)
- **Infrastructure:** [Docker](https://www.docker.com/) & [Nginx](https://nginx.org/)
- **Deployment:** Containerized multi-stage builds for optimized production delivery.

## 🚀 Getting Started

### Prerequisites
- Node.js (LTS)
- Docker & Docker Compose

### Local Development
1. Clone the repository:
   ```bash
   git clone https://github.com/chandan8969/claude-repo.git
   cd claude-repo
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
   Visit `http://localhost:4321` in your browser.

### Production Deployment (Docker)
To launch the full stack (Frontend + Backend):
```bash
docker-compose up -d --build
```
The site will be available via Nginx at the configured port.
